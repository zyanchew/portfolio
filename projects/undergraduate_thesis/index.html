Bachelor's Graduation Project

📍Title: Dynamic Accessibility Measurement Using Human Mobility Big Data

This project was conducted as part of my bachelor’s graduation thesis, focusing on analyzing human mobility data to evaluate accessibility changes during extreme events. By examining mobility patterns before, during, and after a hurricane, this study aims to reveal community resilience and recovery trends.

📌 Key Takeaway
This study demonstrates how mobility data and data-driven approaches can offer valuable insights into infrastructure resilience and inform disaster preparedness strategies.

📈 Results

  • Changes in Accessibility
    Significant fluctuations in accessibility metrics before and after the hurricane event.

  • Community Resilience & Recovery Patterns
    Clustering reveals differentiated recovery responses across communities.

Three types of community response patterns with significant differences were found, including positive responses, stable responses, and negative responses.

📊 Data

The dataset was sourced from SafeGraph, containing anonymized weekly visits to Points-of-Interest (POIs). The study area is Jacksonville, Florida, and the extreme event is Hurricane Dorian.

🔍 Methodology

  1. Data Preprocessing: Cleaning and organizing raw data.
  2. Accessibility Measurement: Quantifying accessibility for each community using POI visit data.
  3. Trend Analysis: Applying agglomerative clustering to detect community response and recovery patterns.
  4. Spatial Visualization: Generate map using QGIS.

Accessibility Metric:

Accessibility metric captures how accessible a service (e.g., gas stations) is to a community, based on POI visits and proximity. A distance decay function adjusts for travel sensitivity to distance.

Accessibility Changes:

To assess community response to the hurricane event, accessibility changes are computed across different periods. These changes are then clustered to identify distinct patterns in resilience and recovery. To better identify the communities at risk, a map is created for spatial visualisation.
